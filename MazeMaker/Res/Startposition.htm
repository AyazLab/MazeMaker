Home > Adding the Start Position

To set the starting position, select the "Start Pos" tool and click anywhere in the pane to set the starting point.

Since there can be only one stating position, clicking again simply sets the start position to the new location.

Initially, the start position faces to the right, but the direction can be changed. First, select the object (for assistance, see Editing a maze.) and go to "Rotate" under the "Edit" menu, or er the "Angle" parameter under the start position's properties.

Note: When saving a maze without a start position, an alert will appear. Mazes can be saved without start positions, but the default starting location will be from the top-left corner of the editing pane, facing to the right.
