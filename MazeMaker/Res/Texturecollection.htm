Home > Textures > The Texture Collection

In order to apply textures, you must have images available for texturing. The Texture Collection is a collection of images available for use in the maze.

To add a texture, find the "Texture Collection" (under "Collections" toolbar). Then you can add images to the collection from three locations:

  • the Standard Library (by default "C:\Program Files\Maze Suite\Library"),
  • the User Library (by default "My Documents\Maze Suite\Library"),  or
  • the same folder
    • that contains your maze file.

If the image file exists somewhere else on the computer, it must be copied into one of these locations.

Once textures have been added to the Texture Collection, they can be applied to surfaces in the maze. See Applying Textures.
