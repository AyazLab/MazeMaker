Home > Adding Floors

To create floors, select the "Floor" tool and click two locations in the Maze Editor pane to define a rectangular floor between them.

To build another floor, repeat the process. To stop making floors, select the "Pointer" tool or press the Escape key.

When you create a floor, a ceiling is also created directly above it automatically. To turn this off, exit the "Floor" tool and select the object (for assistance, see Editing a maze.) At the bottom of the floor's options, change the "Ceiling" property to "false".
