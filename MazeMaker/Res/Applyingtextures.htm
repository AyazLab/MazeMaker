Home > Textures > Applying Textures

Once you have added a texture to the Texture Collection, you can apply it to any wall, ceiling, or floor. Once you have selected the desired object (see Selecting Objects for assistance), find the parameter for Texture (for walls), TextureFloor, or TextureCeiling (in the Floor object, if "Ceiling" is set to True). From here you can select any textures already in the Texture Collection.

Within the selected object's properties, the texture's display settings can be adjusted. The aspect ratio, for tiled textures, affects the shape of the tiled image. To find the aspect ratio that is native to the texture image, expand (click the "+") the Texture parameter then expand the "Image" parameter within that. Divide Width by Height.
Also available is the option to rotate the texture.
