MazeMaker has a graphical user interface that allows you to sketch maps visually with the mouse.

Near the top of the MazeMaker window are a series of toolbars, which are used to select tools and perform certain actions. A pane on the right is used to display or modify properties of objects.

To get started, select "New" to start from scratch, or "Open" to load a pre-existing maze.

You can create maze lists using MazeListBuilder under the "Tools" menu.
