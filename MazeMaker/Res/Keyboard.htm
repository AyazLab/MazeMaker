KeysFunction
Ctrl + O Open Files
Ctrl + N Start New Maze
Ctrl + S Save the current file
Ctrl + C Copy selected object
Ctrl + V Paste copied object
Delete Delete selected object
Ctrl + W Change to Wall Mode
Ctrl + F Change to Floor Mode
Ctrl + L Change to Light Mode
Ctrl + P Change to Start Point Mode
Ctrl + E Change to End Region Mode
Ctrl + M Change to Static Model Mode
Ctrl + D Change to Dynamic Model Mode
Escape Change to Pointer Mode
Ctrl + Left Cursor Move the currently selected item to left in the maze
(Mouse should be in "pointer" mode)
Ctrl + Right Cursor Move the currently selected item to Right in the maze
(Mouse should be in "pointer" mode)
Ctrl + Up Cursor Move the currently selected item to Up in the maze
(Mouse should be in "pointer" mode)
Ctrl + Down Cursor Move the currently selected item to Down in the maze
(Mouse should be in "pointer" mode)
Shift + Space Cycle through all items in the maze. Good for selecting items easily
(Mouse should be in "pointer" mode and initially an item should be selected)
